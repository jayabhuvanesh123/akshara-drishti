About the Project

Research Overview

Akshara-Drishti is a research initiative focused on recognizing ancient Indic scripts using computer vision and machine learning techniques.

The project aims to help researchers, historians, and students interpret inscriptions found in temples, monuments, and archaeological artifacts.

Process

How It Works

Step 1

User uploads an image of an inscription.

Step 2

The system preprocesses the image using digital restoration techniques.

Step 3

A convolutional neural network analyzes the script pattern.

Step 4

The system predicts the script type and confidence score.
