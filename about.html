About the Project

Research Overview

Akshara-Drishti is an advanced AI research initiative focused on recognizing historical Indic scripts using deep learning. It actively supports Brahmi, Devanagari, Tamil, Telugu, Kannada, and Malayalam.

By bridging synthetic font-generation datasets with few-shot learning on real archaeological data, the project overcomes the severe "domain gap" between crisp modern texts and highly degraded ancient stone inscriptions.

Methodology

Dual-Pipeline Inference

Modern Path

Evaluates 0°, 90°, 180°, and 270° rotations to handle handwritten or phone-captured printed text globally.

Inscription Path

Applies CLAHE contrast enhancement and edge sharpening before extracting multi-stride geometric patches.

Entropy Math

Calculates mathematically normalized Shannon Entropy to compare confidence paths and discard confused patches.

Hard Voting

Resolves Grantha-derived letterform clashes (Tamil vs Kannada) using cubic confidence weighting & tiebreakers.
